Privacy Policy

ViralClipAgent is a private workflow tool used to process video content, generate short clips, captions, and upload-ready packages.

Data We Process

The tool may process video files, transcripts, captions, hashtags, clip metadata, and upload status information provided by the user.

TikTok Access

If TikTok integration is enabled, ViralClipAgent uses TikTok authorization only to allow the authorized user to upload finalized clips as drafts for review.

Data Sharing

ViralClipAgent does not sell user data. Content is only used for the user's own workflow and upload preparation.
